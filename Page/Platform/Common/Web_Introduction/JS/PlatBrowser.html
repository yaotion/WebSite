PlatBrowser.js

使用地址：<script src="/_JScripts/Platform/Common/PlatBrowser.js"></script>
使用说明：系统浏览器相关方法

  • GetRequest()

    功能：获取url参数
    例子：
      var Request = new Object();
      Request = GetRequest();
      var pageid=Request['pageid'];

  • getPluginName()

    功能：获取浏览器安装的所有插件的名称
    返回值：浏览器安装的插件名称
    例子：
      $("#result").html(getPluginName());
    结果：Widevine Content Decryption Module;Shockwave Flash;Chrome PDF Viewer;Native Client;Chrome PDF Viewer;

  • checkePlugs(pluginname)

    功能：检查是否安装了某插件，如果安装了返回版本号
    输入参数：
      pluginname：插件名称
    返回值：有则返回版本号 否则返回false
    例子：
      $("#result").html(checkePlugs("Shockwave Flash").toString());
    结果：25.0 r0

  • isIe()

    功能：判断是否IE
    返回值：是IE浏览器 返回true 否则 返回 false
    例子：
      $("#result").html(isIe().toString());
    结果：true

  • isFireFox()

    功能：判断是否firefox
    返回值：是火狐浏览器 返回true 否则 返回 false
    例子：
      $("#result").html(isFireFox().toString());
    结果：true

  • cpuInfo()

    功能：获取CPU 信息
    使用限制：仅适用于IE浏览器，目前不可用，如需使用此方法，请去掉注释信息，并在工具-->Internet选项-->安全-->自定义级别 -->启用"对没有标记为安全的ActiveX控件进行初始化和脚本运行"
    返回值：CPU信息
    例子：
      $("#result").html(cpuInfo().toString());
