使用地址：<script src="/_JScripts/Platform/Common/PlatDialogMessage.js"></script>
使用说明：弹框相关方法

  • Message(strContent, title)

    功能：页面输出错误提示
    例子：
      Message("输入错误", "提示");

  • artDialogOpen(url,strTitle,width,height,lock)

    功能：打开对话框
    输入参数：
      url：url地址（string）
      strTitle：标题（string）
      width：宽（int）
      height：高（int）
      lock：是否锁屏（true 锁屏，false 不锁屏）
    例子：
      artDialogOpen("localhost","添加",250,300,true)

  • windowopen(url)

    功能：打开一个新窗口
    输入参数：
      url：地址
    例子：
      windowopen("localhost");

  • openFullWindow(url, winname)

    功能：全屏打开新窗口
    传入参数：
      url：地址
      winname：窗口名称
    例子：
      openFullWindow("localhost","查看");

  • ajaxLoading()

    功能：加载提示窗口显示，依赖easyui样式
    例子：
      ajaxLoading();

  • ajaxLoadEnd()

    功能：加载提示窗口关闭，依赖easyui样式
    例子：
      ajaxLoadEnd();
