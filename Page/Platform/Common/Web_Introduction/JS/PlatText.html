使用说明：字符串相关方法

  • String.prototype.format

    功能：字符类型扩展format方法
    例子：
      var str="我是{0}，今年{1}岁了。";
      var result=str.format("张三",28);
      $("#result").html(result);
    结果：我是张三，今年28岁了。

  • cutcomma(str, nt, fh)

    功能：去掉字符串首尾符号
    输入参数：
      str：字符串
      nt：需要去掉的符号占的位数
      fh：去掉的符号
    返回值：去掉符号后的字符串
    例子：
      var str=",张三,李四,";
      var result=cutcomma(str,1,",");
      $("#result").html(result);
    结果：张三,李四

  • toCutWordWrap(strWord, intLen)

    功能：字符串按限定长度自动换行
    输入参数：
      strWord：字符串
      intLen：换行长度
    返回值：已自动换行的字符串
    例子：
      var str="2017-01-01 10:00:00";
      var result=toCutWordWrap(str,10);
      $("#result").html(result);
    结果：2017-01-01
        10:00:00

  • String.prototype.trim

    功能：字符类型扩展trim方法(去掉首尾空格)
    例子：
      var str="   张三   ";
      var result=str.trim();
      $("#result").html(result);
    结果：张三

  • String.prototype.ltrim

    功能：字符类型扩展ltrim方法(去掉左侧空格)
    例子：
      var str="   张三   ";
      var result=str.ltrim();
      $("#result").html(result);
    结果：张三   

  • String.prototype.rtrim

    功能：字符类型扩展rtrim方法(去掉右侧空格)
    例子：
      var str="   张三   ";
      var result=str.rtrim();
      $("#result").html(result);
    结果：   张三
