使用说明：日期相关方法

  • Date.prototype.fmttt

    功能：给Date增加fmttt属性
    例子：
      $("#result").html(new Date().fmttt("yyyy"));
    结果：2017

  • setTime(str, fmtt)

    功能：将日期字符串格式化为fmtt格式日期返回
    输入参数：
      str：日期字符串
      fmtt：格式化样式
    返回值：格式化后的日期字符串
    例子：
      $("#result").html(setTime("2017-01-01 10:59:59","yyyy-MM"));
    结果：2017-01

  • compareCalendar(startDate, endDate)

    功能：比较时间入口 时间字符串格式为 “2017-01-01”或 “2017-01-01 01:00:00”
    输入参数：
      startDate：开始时间
      endDate：结束时间
    返回值：开始时间小于结束时间 返回true 否则 返回 false
    例子：
      $("#result").html(compareCalendar("2017-01-01 10:00:00","2017-01-01 10:59:59")?"true":"false");
    结果：true

  • compareDate(checkStartDate, checkEndDate)

    功能：比对不带时间的日期字符串 时间字符串格式为 “2017-01-01”
    输入参数：
      checkStartDate：开始日期
      checkEndDate：结束日期
    返回值：开始日期小于结束日期 返回true 否则 返回 false
    例子：
      $("#result").html(compareDate("2017-01-01","2017-02-01")?"true":"false");
    结果：true

  • compareTime(startDate, endDate)

    功能：比对带时间的时间字符串 时间字符串格式为 “2017-01-01 10:59:59”
    输入参数：
      startDate：开始时间
      endDate：结束时间
    返回值：开始时间小于结束时间 返回true 否则 返回 false
    例子：
      $("#result").html(compareTime("2017-01-01 10:00:00","2017-01-01 10:59:59")?"true":"false");
    结果：true

  • MinutesToChineseHM(nMinutes)

    功能：分钟数转换为中文的时分 例如 例如 200分钟 返回3时20分
    输入参数：
      nMinutes：分钟数
    返回值：时分
    例子：
      $("#result").html(MinutesToChineseHM(200));
    结果：03时20分

  • SecondToChineseDHM(nSeconds)

    功能：秒数转换为中文的天时分 例如 86400秒 返回1天0时0分
    输入参数：
      nSeconds：秒数
    返回值：天时分
    例子：
      $("#result").html(SecondToChineseDHM(86400));
    结果：1天0时0分

  • SecondToHHmmss(nSeconds)

    功能：秒数转换为时分秒 例如 36000秒 返回10:00:00
    输入参数：
      nSeconds：秒数
    返回值：时分秒
    例子：
      $("#result").html(SecondToHHmmss(36000));
    结果：10:00:00

  • StrFormatReturnDate(str)

    功能：将日期字符串格式化日期返回
    输入参数：
      str：日期字符串
    返回值：日期
    例子：
      $("#result").html(StrFormatReturnDate("2017-01-01 10:00:00").toString());
    结果：Sun Jan 01 2017 10:00:00 GMT+0800 (中国标准时间)

  • getdate()

    功能：获取当前日期yyyy-mm-dd
    返回值：日期
    例子：
      $("#result").html(getdate().toString());
    结果：2017-05-09 10:00:18
