SystemLogic.js

使用地址：<script src="/_JScripts/Platform/Common/SystemLogic.js"></script>
使用说明：系统通用相关方法

  • SetHeightWidth()

    功能：重设datagrid宽高
    说明：
    1. 默认重设easyui-datagrid 的 id 为 list的宽高
    2. 增删改查所在的区域div的 id 为 SerachDiv
    例子：
      SetHeightWidth();

  • SaveLog(nType, strPageUrl, strPageName)

    功能：记录日志
    输入参数：
      nType：类型 1-访问、2-新增、3-删除、4-修改、5-查询
      strPageUrl：页面url
      strPageName：页面名称
    例子：
      SaveLog(1,"localhost","测试");

  • publicDelete(url, content)

    功能：公用删除easyui-datagrid 的 id为list可自动刷新
    输入参数：
      url：地址
      content：提示内容
    例子：
      publicDelete("localhost", "确认删除该项吗？");

  • TreeGridpublicDelete(url, content)

    功能：公用删除easyui-treegrid 的 id为list可自动刷新
    输入参数：
      url：地址
      content：提示内容
    例子：
      TreeGridpublicDelete("localhost", "确认删除该项吗？");

  • publicDeleteReload(url, content)

    功能：公用删除后刷新整个页面
    输入参数：
      url：地址
      content：提示内容
    例子：
      publicDeleteReload("localhost", "确认删除该项吗？");

  • publicShowResult(data)

    功能：easyui-datagrid 的id为list公用删除成功后提示且刷新
    输入参数：
      data：结果"true"
    例子：
      publicShowResult("true");

  • publicTreeGridShowResult(data)

    功能：easyui-treegrid 的id为list公用删除成功后提示且刷新
    输入参数：
      data：结果"true"
    例子：
      publicTreeGridShowResult("true");

  • isNullMessage(id, content, tle, focus)

    功能：判断控件值是否为空
    输入参数：
      id：控件id
      content:提示内容
      tle:标题
      focus:是否获取焦点 false-不获取 否则获取
    例子：
      isNullMessage("tbNumber", "请输入工号", "提醒");

  • ConvertFileSize(fSize, retain)

    功能：转换文件大小接收b返回kb，mb
    输入参数：
      fSize：文件大小
      retain:保留小数点后位数
    例子：
      $("#result").html(ConvertFileSize(5000,2));
    结果：4.88KB

  • bytesToSize(bytes)

    功能：字节转换
    输入参数：
      bytes：字节大小
    例子：
      $("#result").html(bytesToSize(5000));
    结果：4.88KB

  • setFirstPage(ids)

    功能：easyui-datagrid查询后返回第一页
    输入参数：
      ids：控件id
    例子：
      setFirstPage("#list");
